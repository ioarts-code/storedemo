🚀 QUICK START (3 STEPS)

1️⃣  CREATE HYGRAPH SCHEMA
   • Go to hygraph.com
   • Create Service, Category, and Tag models
   • Add these fields:
     - Service: name, description, shortDescription, category, tags, url, icon, image
     - Category: name, slug
     - Tag: name, slug

2️⃣  GET API ENDPOINT
   • In Hygraph: API Access → Endpoints → Copy Content API URL
   • (Optional) Create auth token in API Access → Tokens

3️⃣  CONFIGURE APP
   • pnpm install
   • pnpm dev
   • Click ⚙️ icon → Enter endpoint → Save
   • Done! Services load automatically
